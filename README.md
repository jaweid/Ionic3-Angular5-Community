# 时间买卖社区 App

这是一个使用ionic3开发的类似咸鱼的二手社区app，主要包括寻人，寻事，寻物，二手交易，时间买卖，发布信息，订单等模块。包含了丰富的常用场景，开发中常遇到的页面布局方式。

在登录退出模块展示了在Angular应用里如何使用***RXJS***进行数据状态管理。



# 运行方式

```
npm install

ionic run serve
```
